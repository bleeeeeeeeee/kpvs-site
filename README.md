# KPVS Site

Каталог товаров с корзиной и избранным.

## Быстрый старт

```bash
npm install
npm start
```

Открыть: **http://localhost:3000** (автоматически редирект на /welcome)

## Структура

- `server.js` - Express сервер
- `db.js` - SQLite БД (56 товаров)
- `products.json` - Данные товаров
- `*.html` - Страницы (welcome, mens, womens, product)
- `css/style.css` - Стили (172 строки, 16KB)
- `js/*.js` - Логика

## Функции

✓ Два раздела (мужской/женский)
✓ Корзина и избранное
✓ Фильтры и сортировка
✓ localStorage синхронизация
✓ Модальные окна
✓ Адаптивный дизайн

## API

- `GET /api/products/mens` - товары мужские
- `GET /api/products/womens` - товары женские
- `GET /api/product/:id` - один товар

## Как работает

1. npm start → сервер + БД инициализируется из products.json
2. Фронтенд запрашивает API → загружает товары
3. Добавление в корзину/избранное → сохраняется в localStorage
4. Все данные сохраняются при перезагрузке

## Если не работает

- Проверь порт 3000 свободен
- Удали kpvs.db → перезапусти (БД пересоздается)
- F12 → Console → смотри ошибки
- Убедись products.json в корне проекта

## Технологии

- Node.js + Express 4.18.2
- SQLite3 5.1.6
- Vanilla JS (ES6+)
- CSS3
