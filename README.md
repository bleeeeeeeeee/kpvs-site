# Сайт-витрина ООО "КПВС"

Проект сайта-витрины для компании по производству спецодежды ООО "КПВС" в Бресте.

## Функционал:
- Каталог товаров (мужская и женская спецодежда) с переходом на детальную страницу товара
- Детальная страница товара с описанием, материалами и кнопкой "Узнать цену" (отправка email)
- Система избранных товаров (сохранение в localStorage)
- Фильтры по категориям и сортировка товаров
- Адаптивный дизайн
- Динамическая загрузка товаров из JSON базы данных
- Слайдеры товаров с прогресс-барами и умными градиентами (показываются только при возможности скролла)

## Технологии:
- HTML5
- CSS3 (Flexbox, Media Queries, Transitions)
- JavaScript (Fetch API, localStorage, DOM manipulation)
- JSON для базы данных
