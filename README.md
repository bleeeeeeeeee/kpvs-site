# KPVS Site - Каталог товаров

Полнофункциональное веб-приложение для просмотра и управления каталогом товаров с возможностью добавления в корзину и избранное.

## Возможности

- Каталог товаров для мужского и женского разделов
- Корзина с сохранением товаров в браузере
- Список избранных товаров
- Фильтрация и сортировка товаров
- Адаптивный дизайн
- Автоматическое сохранение состояния (localStorage)
- Отслеживание источника товара (из какого раздела добавлен)

## Требования

- Node.js (версия 14 и выше)
- npm (идет в комплекте с Node.js)

## Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

Команда загрузит все необходимые пакеты, перечисленные в `package.json`.

### 2. Запуск сервера

```bash
npm start
```

Сервер запустится на `http://localhost:3000`

### 3. Открыть в браузере

Откройте браузер и перейдите на `http://localhost:3000/welcome`

## Использование

### Навигация

- **Приветствие** (`/welcome`) - стартовая страница
- **Мужской раздел** (`/mens`) - товары для мужчин
- **Женский раздел** (`/womens`) - товары для женщин
- **Товар** (`/product?id=1`) - подробная информация о товаре

### Функции

#### Добавление в корзину
1. Перейдите в раздел (Мужской или Женский)
2. Наведите курсор на карточку товара
3. Нажмите кнопку "В корзину"
4. Кнопка изменится на "Удалить из корзины"

#### Добавление в избранное
1. На карточке товара нажмите "В избранное"
2. Кнопка изменится на "Удалить из избранного"

#### Просмотр корзины
Нажмите кнопку "Корзина" в шапке страницы, чтобы открыть модальное окно со всеми добавленными товарами с указанием их источника (мужской раздел, женский раздел или страница товара).

#### Просмотр избранного
Нажмите кнопку "Избранное" в шапке, чтобы открыть список понравившихся товаров.

## Технологии

- **Backend**: Node.js + Express.js
- **Database**: SQLite3
- **Frontend**: Vanilla JavaScript + HTML + CSS
- **Storage**: Browser localStorage

## Возможные проблемы

### Сервер не запускается
- Убедитесь, что Node.js установлен: `node -v`
- Убедитесь, что npm установлен: `npm -v`
- Попробуйте удалить папку `node_modules` и файл `package-lock.json`, затем запустите `npm install` заново

### База данных не инициализируется
- Если сервер выдает ошибку о `products.json`, убедитесь что файл находится в корневой папке проекта
- При необходимости удалите файл `kpvs.db` и перезапустите сервер

### Порт 3000 уже занят
- Измените порт в `server.js` (по умолчанию строка: `const PORT = 3000`)
- Или закройте приложение, использующее порт 3000

### Товары не загружаются
- Проверьте браузерную консоль (F12 → Console) на предмет ошибок
- Убедитесь, что сервер работает (должны видеть сообщение в консоли сервера)
