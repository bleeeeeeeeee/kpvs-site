Проект является клиентским веб-приложением и использует загрузку данных из файла products.json. Для корректной работы проект необходимо запускать через локальный HTTP-сервер. Ниже приведены способы запуска проекта локально.

Способ 1. Запуск через Visual Studio Code
	1.	Установить Visual Studio Code на компьютер
	2.	Запустить Visual Studio Code
	3.	Открыть папку с проектом через меню File Open Folder
	4.	Открыть вкладку Extensions
	5.	Найти расширение Live Server и установить его
	6.	После установки перейти в проводнике редактора к файлу welcome.html
	7.	Нажать правой кнопкой мыши на файл welcome.html
	8.	Выбрать пункт Open with Live Server
	9.	Проект автоматически откроется в браузере по локальному адресу

Способ 2. Запуск через Node.js
	1.	Убедиться, что Node.js установлен. Для проверки в терминале ввести команду "node --version"
	2.	Перейти в папку проекта командой cd и указать путь к папке проекта
	3.	Ввести команду "npx serve ." (если появиться сообщение "Need to install the following packages:   serve@14.2.5   Ok to proceed? (y) - надо вести в терминал "y" и нажать Enter)
	4.	Дождаться сообщения о запуске сервера
	5.	Открыть браузер
	6.	Перейти по локальному адресу, который будет указан в терминале
	7.	Открыть страницу welcome.html

    При завершении работы через терминал - надо его активировать и нажать Ctrl + C

Проект не рекомендуется запускать путём открытия HTML-файлов напрямую через проводник, так как браузер блокирует загрузку локальных JSON-файлов, что приводит к некорректной работе приложения.
