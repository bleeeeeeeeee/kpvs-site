# KPVS Site - Полностью функциональное приложение

## Быстрый старт

### Обязательно: Запустить Node.js сервер
```bash
npm start
```
Сервер будет работать на **http://localhost:3000**

## Как открывать страницы

### ✅ Правильно (работает везде):
- http://localhost:3000/welcome - Главная страница
- http://localhost:3000/mens - Мужская одежда
- http://localhost:3000/womens - Женская одежда
- http://localhost:3000/product - Страница товара
- http://localhost:3000/welcome.html - Также работает
- http://localhost:3000/mens.html - Также работает

### ❌ НЕ РАБОТАЕТ через Live Server
Live Server (VSCode) работает на другом порту и не имеет доступа к API. Поэтому товары не загружаются.

**Решение:** Всегда используйте `npm start` и открывайте **http://localhost:3000**

## Функциональность

✅ Загрузка товаров из SQLite базы данных  
✅ Фильтрация по категориям  
✅ Сортировка по названию и ID  
✅ Единая корзина для всех разделов  
✅ Единое избранное для всех разделов  
✅ Полностью рабочие модальные окна  
✅ Email запросы о цене  
✅ Карточки товаров с изображениями  

## Структура проекта

```
/
├── server.js              # Express сервер с REST API
├── db.js                  # SQLite база данных
├── package.json           # Зависимости Node.js
├── kpvs.db               # База данных товаров
├── products.json         # Исходные данные для БД
├── *.html                # HTML страницы
├── css/style.css         # Стили
├── js/                   # JavaScript логика
│   ├── mens.js          # Логика мужского раздела
│   ├── womens.js        # Логика женского раздела
│   ├── product.js       # Логика страницы товара
│   └── welcome.js       # Логика главной страницы
└── img/                 # Изображения
```

## API Endpoints

- `GET /api/products/mens` - Все мужские товары
- `GET /api/products/womens` - Все женские товары
- `GET /api/product/:id` - Товар по ID

## Корзина и Избранное

Оба работают через **localStorage** и синхронизированы между разделами:
- Товар, добавленный в корзину из мужского раздела, видно в женском
- Товар, добавленный в избранное из женского раздела, видно в мужском
- Данные сохраняются и переживают перезагрузку страницы

## Тестирование

Открыте http://localhost:3000/cart-test.html для проверки корзины и избранного.
